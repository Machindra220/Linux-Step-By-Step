# 🥷 06. Vi Editor

> **Goal:** Learn the powerful, "Ninja" editor used by professional Admins.

### 💡 Concept
* **Vi** (or **Vim**) is different because it has **Modes**. You cannot just start typing immediately; you have to tell Vi you want to "Insert" text first.



### 🛠️ The Three Main Modes
1. **Command Mode** (Default): For moving around and deleting.
2. **Insert Mode**: For typing text (Press **`i`** to enter this).
3. **Last Line Mode**: For saving and quitting (Press **`:`**).

**The Secret Codes:**
* **`i`** : Start typing (Insert).
* **`Esc`**: Stop typing (Back to Command Mode).
* **`:w`**: Save (Write).
* **`:q!`**: Quit without saving (The emergency exit!).
* **`:wq`**: Save and Quit.

### 🎮 Mini-Challenge: Level 24 - The Ninja Mode
> "Open a file with `vi test.txt`. Press `i`, type 'Linux is Power', then press `Esc`. Finally, type `:wq` and hit Enter. You’ve survived your first encounter with Vi!"
