# ✍️ Phase 4: Reading and Writing Data

### 🎯 What we are covering:
We are opening the files! We will learn:
* **Quick Looks:** Using `cat`, `head`, and `tail` to peek inside files.
* **The Bookmarks:** Scrolling through long files with `less`.
* **The Pens:** Learning the **Nano** and **Vi** editors to actually write your own scripts and notes.

### ❓ Why it matters:
Everything in Linux is a file. Configuration files, website code, and system logs are all just text. If you can read and edit text files, you can control the entire behavior of the computer.
