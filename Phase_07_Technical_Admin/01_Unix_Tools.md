# 🧰 01. Unix Tools (The Swiss Army Knife)

> **Goal:** Use built-in tools to find files, check the date, and see your disk space.

### 💡 Concept
* Linux comes with small, specialized tools that do one thing perfectly. 
* Whether you need to find a lost file or check if your hard drive is full, these are your "Go-To" gadgets.

### 🛠️ The Commands
`find /home -name "*.txt"`
`df -h`

**Tag Details:**
* **f** - **find**: Searches for files based on names, size, or date.
* **d** - **df**: **"Disk Free"** – shows how much space is left on your drives.
* **h** - **human-readable**: (Used with `df` or `free`) Shows sizes in **GB/MB** instead of just bytes.
* **u** - **uptime**: Shows how long the computer has been running without a restart.
