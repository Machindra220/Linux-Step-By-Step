# 🏗️ Phase 7: The System Admin

### 🎯 What we are covering:
You are now the Admin! We will learn:
* **Health Check:** Monitoring memory (`free`), disk space (`df`), and running apps (`top`).
* **Networking:** How to talk to other computers and download from the web (`ping`, `wget`).
* **Automation:** Using **Crontab** to make the computer do chores while you sleep.

### ❓ Why it matters:
A good Linux Admin doesn't work hard; they work smart. By learning to monitor and automate the system, you make the computer work for you, rather than you working for the computer.
