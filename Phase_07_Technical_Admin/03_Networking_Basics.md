# 🌐 03. Networking Basics

> **Goal:** Learn how your Linux computer talks to the rest of the world.

### 💡 Concept
* Linux is the king of the internet. These tools help you check your connection and download files from websites.

### 🛠️ The Commands
`ping google.com`
`wget [URL]`

**Tag Details:**
* **c** - **count**: (Used with `ping`) Tells the computer to stop after 4 pings instead of going forever.
* **i** - **ip**: (Using `ip addr`) Shows your computer's unique **"Digital Address"** on the network.
* **O** - **output-document**: (Used with `wget`) Allows you to save a downloaded file with a **new name**.
* **q** - **quiet**: (Used with `curl` or `wget`) Downloads files without showing all the progress text.



### 🎮 Mini-Challenge: Level 39 - The Signal Tester
> "Run `ping -c 3 google.com`. If you see '3 packets transmitted, 3 received,' your internet is working perfectly! Now, find your IP address using `hostname -I`."
