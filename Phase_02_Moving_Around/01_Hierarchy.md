# 🌳 01. Linux File Hierarchy

> **Goal:** Understand the "Map" of the Linux system. In Linux, everything starts from the Root.

### 💡 The Root Directory (`/`)
* In Windows, you have drives like `C:` or `D:`. In Linux, there are **no drive letters**.
* Everything—every file, every hard drive, and every USB stick—starts from a single point called the **Root**, represented by a forward slash `/`.

### 📂 Key Folders to Know:
* **`/bin`**: Contains **Essential Binaries** (programs like `ls` and `cp`).
* **`/etc`**: The "Control Room." It stores **System Configuration** files.
* **`/home`**: The "Neighborhood." This is where **User Folders** (like yours) are kept.
* **`/root`**: The private home for the **SuperUser** (Admin).
* **`/tmp`**: A place for **Temporary Files** that get deleted when you restart.
