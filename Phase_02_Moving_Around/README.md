# 🗺️ Phase 2: Finding Your Way (The Filesystem)

### 🎯 What we are covering:
Now it's time to type! We will learn:
* **The Map:** How the Linux folder structure (Hierarchy) works.
* **Navigation:** Using `pwd` to find yourself and `cd` to "teleport" between folders.
* **Visibility:** Using `ls` to see what’s hidden and what's not.

### ❓ Why it matters:
In Linux, there are no icons or "My Computer" windows by default in the terminal. Learning to navigate using text is like learning to read a map; once you know the coordinates, you can go anywhere without getting lost.
