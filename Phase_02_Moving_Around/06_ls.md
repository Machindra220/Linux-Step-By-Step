# 👁️ 06. ls (List)

> **Goal:** See what is inside a folder.

### 💡 Concept
* **ls** lists the files and folders in your current location. It’s like turning on the light in a dark room to see what’s on the floor.

### 🛠️ Common Tags
`ls -al`

**Tag Details:**
* **a** - **all**: Shows **hidden files** (files that start with a dot `.`).
* **l** - **long**: Shows **extra details** like file size, owner, and date.
* **h** - **human-readable**: (Used with `-l`) Shows file sizes in **KB, MB, or GB** instead of just bytes.
* **R** - **Recursive**: Shows files in the current folder AND all **sub-folders**.

### 🚀 Try This:
`ls -lh`
* This shows your files in a detailed list with sizes that are easy to read!
