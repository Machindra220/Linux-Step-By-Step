# 🚶 03. cd (Change Directory)

> **Goal:** Move from one folder to another.

### 💡 Concept
* **cd** is the command used to **navigate** or "teleport" through folders.

### 🛠️ The Command
`cd [target_directory]`

**Useful Shortcuts:**
* `cd ..`
  * **.** **.** : Move **Up** one level (to the parent folder).
* `cd ~`
  * **~** : Go directly to your **Home** folder.
* `cd -`
  * **-** : Go back to the **previous** folder you were just in.

### 🚀 Practice
If you are in `/home` and want to go to `/home/Downloads`:
`cd Downloads`
