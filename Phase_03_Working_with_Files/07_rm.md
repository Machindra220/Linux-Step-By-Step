# 🗑️ 07. rm (Remove)

> **Goal:** Delete files and folders.

### ⚠️ WARNING
* There is **no Recycle Bin** in the Linux terminal. Once you delete it with `rm`, it is gone forever!

### 🛠️ The Command
`rm -rf folder_or_file`

**Tag Details:**
* **r** - **recursive**: Deletes a folder and **everything** inside it.
* **i** - **interactive**: Asks "Are you sure?" for every single file. (Great for beginners!).
* **f** - **force**: Tells Linux "Don't ask me questions, just delete it." (Be very careful with this!).

### 🎮 Mini-Challenge: Level 17 - The Destroyer
> "Use `rm -i secret_code.txt`. When the computer asks if you are sure, type `y` for Yes. Feel the power (and the responsibility) of the delete command!"
