# 🏗️ 01. mkdir (Make Directory)

> **Goal:** Learn how to create new folders to stay organized.

### 💡 Concept
* **mkdir** stands for **Make Directory**. 
* It is like taking an empty cardboard box and labeling it for your toys or books.

### 🛠️ The Command
`mkdir -p path/to/folder`

**Tag Details:**
* **m** - **mkdir**: The main command to create a folder.
* **p** - **parents**: This "Super Power" tag allows you to create a folder inside another folder that doesn't exist yet (it creates the whole path).
* **v** - **verbose**: Tells you exactly what Linux just created so you can be sure it worked.

### 🎮 Mini-Challenge: Level 11 - The Architect
> "Create a folder named `Kingdom`. Inside `Kingdom`, create a folder named `Castle`, and inside `Castle`, create a folder named `Treasure`. 
> **Bonus:** Can you do it all with just **one** command using the `-p` tag?"
