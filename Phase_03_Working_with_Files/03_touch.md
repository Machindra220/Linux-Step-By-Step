# ✍️ 03. touch

> **Goal:** Create new, empty files instantly.

### 💡 Concept
* The **touch** command is the fastest way to create a new file. 
* It's like grabbing a fresh, blank piece of paper from a stack.

### 🛠️ The Command
`touch -c filename.txt`

**Tag Details:**
* **t** - **touch**: The command to create or update a file.
* **c** - **no-create**: Tells Linux NOT to create a new file if it already exists (it just updates the time).
* **a** - **access time**: Only changes the "last accessed" time of the file.
