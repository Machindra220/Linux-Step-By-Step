# 🧹 02. rmdir (Remove Directory)

> **Goal:** Delete empty folders you no longer need.

### 💡 Concept
* **rmdir** stands for **Remove Directory**.
* **Safety First:** This command **only** works if the folder is completely **empty**. This prevents you from accidentally deleting important files!

### 🛠️ The Command
`rmdir folder_name`

**Tag Details:**
* **r** - **rmdir**: The command to remove an empty directory.
* **p** - **parents**: Removes the folder and its parent folders if they also become empty.
