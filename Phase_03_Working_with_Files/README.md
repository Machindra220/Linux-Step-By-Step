# 🛠️ Phase 3: Working with Files

### 🎯 What we are covering:
It’s time to build! We will learn:
* **Creation:** Making new folders (`mkdir`) and empty files (`touch`).
* **Organization:** Copying (`cp`), moving (`mv`), and renaming files in bulk.
* **Cleanup:** How to safely delete things (`rm`)—remember, there is no "Recycle Bin" here!

### ❓ Why it matters:
Managing data is the core of any IT job. By mastering these commands, you learn how to organize thousands of files in seconds, something that would take hours using a mouse and keyboard.
