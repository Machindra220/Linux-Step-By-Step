# 📦 05. mv (Move)

> **Goal:** Move files to a new home or rename them.

### 💡 Concept
* **mv** stands for **Move**. 
* Unlike `cp`, this command picks up the original and puts it somewhere else. 
* **Fun Fact:** Linux also uses `mv` to **Rename** files (it's like moving a file from an old name to a new name).

### 🛠️ The Command
`mv -i source destination`

**Tag Details:**
* **m** - **mv**: The move command.
* **i** - **interactive**: Asks for permission before overwriting a file at the destination.
* **u** - **update**: Only moves the file if it is newer than the one already at the destination.
