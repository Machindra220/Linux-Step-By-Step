# 📑 04. cp (Copy)

> **Goal:** Make a duplicate of a file or folder.

### 💡 Concept
* **cp** stands for **Copy**. 
* It’s like using a Xerox machine. The original stays where it is, and a new identical version is created somewhere else.



### 🛠️ The Command
`cp -r source_file destination`

**Tag Details:**
* **c** - **cp**: The copy command.
* **r** - **recursive**: **CRITICAL!** Use this if you want to copy a whole folder and everything inside it.
* **i** - **interactive**: Asks you "Are you sure?" before overwriting an existing file.
* **v** - **verbose**: Shows you exactly which files are being copied.
